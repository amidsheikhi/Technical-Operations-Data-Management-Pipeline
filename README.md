# Automated Technical Data Management and Reporting Pipeline
An industrial engineering case study deploying a relational MS Access database and an automated Power BI KPI dashboard for infrastructure asset tracking.

## Project Overview
This repository contains the source development files for an integrated operational data solution. The project replaces manual tabular logging workflows with an automated data pipeline utilizing industrial engineering methodologies.

## Repository Contents
* **`*.accdb`**: Microsoft Access Relational Database containing the 3NF data schema, tables, and calculated query processing layer (`qry_LeadTime_Compliance`).
* **`*.pbix`**: Microsoft Power BI Desktop Project File containing the interactive KPI visual configurations and trend line analytics.

## Core KPIs Tracked
1. Average Maintenance Lead Time (Days)
2. Service Level Agreement (SLA) Deadline Compliance Rate (%)
3. Continuous Volumetric Flow Tracking Trends (m³/h)
